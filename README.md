# Automacao_Residencial
Automacao Resisdencial - Robotica PMC/SATC

Projeto visando a comodidade do ser humanos, onde uma planta genérica de uma residência foi construída e os comandos da porta, ventilação e iluminação são realizador de forma remota por um controle IR, palmas e sensores de temperatura e umidade. Sendo todos os dados apresentados em uma tela de cristal líquido.
